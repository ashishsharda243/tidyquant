NLP DATA Mining Warehouse

NLP Model for Text Mining

This exciting project was all about building an in-house data warehouse through remote access to client’s server for the purpose of developing and deploying of a text mining model (Natural Language Processing (NLP) solution) that extracts real time data from different online sources (social media & online publications) and provides insights & trends over an interactive dashboard. Built entirely over inhouse data warehouse, the solution is aptly scalable to manage & maintain consistently incrementing data.

Technologies used

  • Python – Version 3.7.3 (library packages: – Pandas, NumPy, SciPy, Scikitlearn etc.)
  • Citrix software for remote access
  • Apache Hadoop – Version – 3.1.1
  • Anaconda Server – Version Anaconda 2019.0
  • TensorFlow – Version r1.13
  • NLTK Package – Version 3.4.1
  • Power BI Server – Version 15.0.1102.299
